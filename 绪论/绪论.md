# 核力及核势垒

核力强相互作用、短程、饱和、**极短程内有排斥芯**、与自旋有关、有非中心力部分(同向自旋线列吸引、并列排斥)、自旋和轨道角动量关系也影响核力

$\text{库伦势垒公式 }V(R)=\frac{e^2}{4\pi\varepsilon_0}\frac{Z_1Z_2}{r_0(A_1^{\frac{1}{3}}+A_2^{\frac{1}{3}})}$

**中子没有库伦势垒**(能量低也能进入原子核,但是几率小:核外波长太长,进入核内有能量限制波长大幅降低,为了平滑过渡使得频率大幅增加、幅度大幅降低,幅度与概率相关.故为了产生核反应,只能利用共振(调入射到原子核时的频率)提高幅度来提高概率)

**势垒对衰变/核反应射出带电粒子和核反应射入带电粒子都有阻碍作用!**

# 原子核的宇称

**宇称**就是波函数的对称性

![alt text](image-17.png)
==本征态==:被某个算符作用以后,本身不变,只是多乘了一个数的态(或者波函数)  
==本征值==:多乘的这个数就是本征值(例如这里的$\pi$,正负决定了奇偶宇称态)

**空间反演条件下物理规律不变**(空间反演就是(x,y,z)-->(-x,-y,-z))
